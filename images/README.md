# Images Folder

Place your wedding photos here:

- `bride.jpg` - Photo of the bride
- `groom.jpg` - Photo of the groom
- Add any additional photos for the gallery

## Recommended Image Sizes

- **Couple Photos**: 400x500 pixels (portrait orientation)
- **Gallery Images**: 1200x800 pixels (landscape orientation)

## Supported Formats

- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)

---

**Note**: Make sure to update the image paths in `index.html` if you use different filenames.
